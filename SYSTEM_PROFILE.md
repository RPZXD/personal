# โปรไฟล์ระบบบริหารจัดการงานบุคลากร (Personal HRIS)

## ภาพรวมของระบบ (System Overview)
ระบบบริหารจัดการงานบุคลากร (Personal Management System) เป็นระบบวิสารสนเทศที่ถูกพัฒนาขึ้นเพื่อใช้ในการจัดเก็บและบริหารจัดการข้อมูลบุคลากรภายในหน่วยงานอย่างเป็นลำดับขั้นตอน ช่วยลดความซับซ้อนในการทำเอกสาร และสามารถตรวจสอบข้อมูลย้อนกลับได้อย่างมีประสิทธิภาพ

---

## คุณสมบัติหลัก (Key Features)

### 1. ระบบจัดการข้อมูลส่วนบุคคลและโปรไฟล์ (Profile Management)
*   **บันทึกทะเบียนประวัติ**: จัดเก็บข้อมูลพื้นฐาน, ข้อมูลติดต่อ และประวัติการทำงานของบุคลากร
*   **Dashboard สรุปผล**: แสดงภาพรวมการปฏิบัติงานส่วนตัว เช่น จำนวนวันลา, จำนวนชั่วโมงการฝึกอบรม และผลงาน/รางวัลที่ได้รับ

### 2. ระบบจัดการการลา (Leave Management System)
*   **บันทึกการลา**: รองรับการลาประเภทต่างๆ (ลาป่วย, ลากิจ, ลาพักผ่อน ฯลฯ)
*   **ระบบตรวจสอบสิทธิ์**: ตรวจสอบจำนวนวันลาคงเหลือตามสิทธิ์ของบุคลากรแต่ละประเภท
*   **พิมพ์ใบลา**: สามารถพิมพ์ใบลาในรูปแบบทางการ (PDF/Print) เพื่อเสนอลงนามได้ทันที (`print_leave.php`)
*   **สรุปยอดการลา**: แสดงรายงานการลาแบบรายเดือนและรายปี เพื่อใช้ในการประเมินผลการปฏิบัติงาน

### 3. ระบบบันทึกประวัติการพัฒนาบุคลากร (Training & Development)
*   **บันทึกการฝึกอบรม/สัมมนา**: จัดเก็บข้อมูลการพัฒนาตนเอง วันที่อบรม สถานที่ และจำนวนชั่วโมง
*   **ระบบสรุปรายเดือน**: แสดงรายงานการอบรมรวมในแต่ละเดือน (`training_monthly.php`)
*   **สรุปรายงานการอบรม**: พิมพ์รายงานสรุปการเข้าอบรมทั้งหมดเพื่อใช้เป็นหลักฐานในการเลื่อนวิทยฐานะหรือประเมินผล

### 4. ระบบจัดการรางวัลและผลงาน (Awards & Achievements)
*   **บันทึกเกียรติประวัติ**: จัดเก็บข้อมูลรางวัลที่ได้รับ ระดับของรางวัล (โรงเรียน, เขตพื้นที่, ชาติ ฯลฯ)
*   **Portfolio ระบบดิจิทัล**: เป็นศูนย์รวมผลงานที่สามารถเรียกดูและนำไปใช้งานในการทำพอร์ตโฟลิโอได้ทันที

### 5. ระบบสำหรับเจ้าหน้าที่และผู้บริหาร (Officer & Admin Module)
*   **ตรวจสอบการปฏิบัติราชการ**: ดูข้อมูลมาสาย-กลับก่อน (`late_early.php`)
*   **สรุปภาพรวมบุคลากรทั้งหน่วยงาน**: ดูสถิติการลาและการอบรมของบุคลากรทุกคนในระบบ
*   **การจัดการข้อมูลบุคลากร**: เพิ่ม แก้ไข หรือระงับสิทธิ์การเข้าใช้งานของบุคลากรในสังกัด

---

## เทคโนโลยีที่ใช้ (Technology Stack)
*   **Frontend**: HTML5, Modern UI Design, Responsive Web Design สำหรับการใช้งานผ่านมือถือและแท็บเล็ต
*   **Backend**: PHP (MVC Architecture)
*   **Database**: MySQL/MariaDB
*   **API**: มีระบบ API สำหรับรองรับการเชื่อมโยงข้อมูลกับระบบอื่นๆ

---

## จุดเด่นของระบบ (Highlights)
*   **Self-Service**: บุคลากรสามารถจัดการข้อมูลและพิมพ์เอกสารด้วยตนเองได้ทุกที่ทุกเวลา
*   **Accurate Reporting**: มีระบบสรุปผลอัตโนมัติที่ลดความผิดพลาดในการคำนวณวันลาหรือชั่วโมงอบรม
*   **Official Document Support**: รองรับการพิมพ์แบบฟอร์มที่เป็นทางการตามระเบียบงานสารบรรณ
*   **Role-based Access Control**: ระบบความปลอดภัยแบ่งระดับการเข้าถึงข้อมูลตามบทบาทหน้าที่ (Admin, Officer, Teacher)
